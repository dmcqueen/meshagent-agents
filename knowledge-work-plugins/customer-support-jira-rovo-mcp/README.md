# Jira Rovo Mcp

## Summary

Expose Atlassian Rovo MCP for Jira and Jira Service Management

## Description

This agent lets users and teams interact with an Atlassian Rovo MCP integration for Jira and Jira Service Management to resolve customer support workflows and operational questions. It can turn ambiguous requests into concrete, actionable steps, help craft and refine tickets, and provide guidance tailored to specific goals, constraints, and context so teams can move work forward efficiently. By answering domain-specific questions about customer support processes and tooling, it supports faster issue resolution, clearer handoffs, and more consistent use of Jira-related features. The agent is designed to streamline daily support operations and improve outcomes for customers and internal stakeholders.
