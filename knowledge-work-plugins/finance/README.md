# Claude Finance

## Summary

An agent that uses Anthropic's finance knowledge work plugin

## Description

This agent helps finance teams analyze questions, summarize reports, and highlight the most decision-relevant signals so stakeholders can act with clarity and speed. It prepares structured outputs for planning, close management, audits, and stakeholder updates, and supports tasks such as preparing journal entries, performing reconciliations, conducting variance analysis, and reviewing financial statements. Users can rely on it to produce clear explanations of implications and concise, actionable summaries that streamline review cycles and audit preparations. When needed, it can generate ready-to-use summaries and checklists to support financial close and compliance workflows.
