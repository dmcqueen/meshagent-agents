# Claude Finance

## Summary

An agent that uses Anthropic's finance knowledge work plugin

## Description

This agent helps finance teams analyze financial questions, summarize reports, and prepare structured outputs that support planning, close activities, and stakeholder updates. Users can get clear explanations of implications from financial analyses, concise highlights of decision-relevant signals in statements or reports, and prepared materials for audit support, reconciliation, journal entry preparation, variance analysis, and close management. It produces organized, actionable deliverables that streamline review cycles and help teams make informed decisions during month-end and audit processes. The agent runs as a finance-focused assistant tuned to deliver concise, work-ready outputs for accounting and financial reporting tasks.
