# Claude Customer Support MeshAgent

**Overview**
An agent that uses Anthropic's customer support knowledge work plugin with a nightly docs sync worker

**Details**
- Agents: claude-customer-support-meshagent (ChatBot), claude-customer-support-meshagent (MailBot), claude-customer-support-meshagent (Worker)

- Skills: customer-research, escalation, knowledge-management, response-drafting, ticket-triage
