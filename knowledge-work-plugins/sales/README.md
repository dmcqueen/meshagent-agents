# Claude Sales

## Summary

An agent that uses Anthropic's sales knowledge work plugin

## Description

This agent helps salespeople and teams streamline prospecting, account research, and deal execution by generating tailored outreach messages, qualification scripts, and briefing materials that reflect customer context and signals. It can summarize customer interactions and market or competitive intelligence to inform call preparation, pipeline reviews, and forecasting conversations. Teams can use it to produce consistent assets for outreach, follow-up, and account strategy, reducing time spent on routine prep and improving alignment across stakeholders. When used with a suitable runtime model, it enables quick iterations on messaging and concise summaries that support better-informed sales decisions.
