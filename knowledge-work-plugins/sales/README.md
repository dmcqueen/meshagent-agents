# Claude Sales

## Summary

An agent that uses Anthropic's sales knowledge work plugin

## Description

This agent helps sales reps and teams prepare targeted prospecting and account messaging, create outreach and follow-up materials, and summarize customer signals to inform deal strategy; users can generate tailored emails, call decks, and qualification scripts to accelerate pipeline progression. It supports account research, call preparation, and competitive intelligence so teams enter conversations with concise insights and actionable next steps. The agent also produces regular briefings and call or pipeline summaries to keep stakeholders aligned and simplify forecasting and pipeline review. Responses are designed to save time on routine sales tasks and improve the consistency and relevance of customer engagement.
