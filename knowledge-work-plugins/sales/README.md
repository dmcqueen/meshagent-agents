# Claude Sales

## Summary

An agent that uses Anthropic's sales knowledge work plugin

## Description

This agent helps sales teams prepare targeted prospecting and account messaging, produce outreach and follow-up materials, and summarize customer signals to guide deal strategy. Teams can use it to research accounts, prepare call briefs and summaries, and generate qualification and competitive-intelligence inputs that improve pipeline progress and conversion decisions. It also supports routine sales operations like daily briefings, forecast and pipeline reviews, and creation of assets needed for outreach. Outputs focus on actionable messaging, clear next steps, and evidence-based recommendations to move opportunities forward.
