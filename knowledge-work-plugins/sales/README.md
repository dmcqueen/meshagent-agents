# Claude Sales

## Summary

An agent that uses Anthropic's sales knowledge work plugin

## Description

This agent helps sales reps and teams prepare targeted prospecting and account messaging, generate outreach and follow-up materials, and summarize customer signals to inform deal strategy. Users can get concise call summaries, tailored qualification prompts, and competitive insights to streamline call preparation and pipeline reviews. It supports creating assets like emails and briefing notes that align with account context, improving consistency and speed across outreach and account management. The agent is intended for practical daily use by sellers and managers to increase efficiency and clarity in sales motions.
