# Claude Engineering Sentry

## Summary

Polling worker that scans unresolved Sentry issues and creates or updates Jira tickets with severity and priority triage.

## Description

This plugin supports engineering sentry workflows and is designed to help users complete practical work with less manual effort. It can poll Sentry for unresolved exceptions on a recurring schedule, deduplicate findings against Jira and create missing bug tickets automatically, and run with a deliberately narrow skill scope focused on incident-response and ticket-triage only. Built-in guidance and reusable skills help keep outputs consistent, context-aware, and action-oriented. It works well for both quick answers and multi-step tasks that require planning, drafting, and refinement. Teams can use it as a day-to-day assistant for engineering sentry decisions, analysis, and communication.
