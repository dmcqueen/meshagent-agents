# Claude Enterprise Search

## Summary

An agent that uses Anthropic's enterprise search knowledge work plugin

## Description

This agent enables teams to find and synthesize relevant internal information from large enterprise knowledge sources to produce concise, decision-ready summaries and context-rich answers grounded in available content. Users can run targeted searches, manage and prioritize sources, and receive synthesized insights that help streamline decision making and reduce time spent locating information. The agent supports crafting search strategies and managing source selection to ensure responses are based on the most pertinent enterprise material. It is best used when you need clear, actionable summaries and answers derived directly from your organizationâ€™s knowledge base.
