# Claude Enterprise Search

## Summary

An agent that uses Anthropic's enterprise search knowledge work plugin

## Description

This agent enables teams to locate and consolidate relevant internal information from large enterprise knowledge sources, turning dispersed documents into concise, decision-ready summaries. Users can ask context-rich questions and receive answers grounded in available enterprise content, with synthesized findings and recommended next steps to support decision making. It helps streamline search strategies and source management so teams spend less time hunting for facts and more time applying insights. The agent is intended for knowledge synthesis and operational workflows where accurate, source-backed answers are required.
