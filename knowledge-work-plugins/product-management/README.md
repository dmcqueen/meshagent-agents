# Claude Product Management

## Summary

An agent that uses Anthropic's product management knowledge work plugin

## Description

This agent helps product teams turn research, feedback, and data into clear, actionable outputs that drive planning and execution. Users can synthesize user research and competitive insights into concise recommendations, draft feature specifications and metrics, and produce prioritized roadmaps that align stakeholders and guide development. It also streamlines regular artifacts like metrics reviews and stakeholder updates so teams maintain alignment and measure progress against goals. By focusing on practical deliverables—specs, briefs, roadmaps, and syntheses—teams can make faster, better-informed product decisions.
