# Claude Product Management

## Summary

An agent that uses Anthropic's product management knowledge work plugin

## Description

This agent helps product teams turn research, feedback, and market signals into clear, actionable plans and communications. Users can synthesize user research into prioritized insights, draft feature specifications and metrics to measure success, and produce concise roadmap and stakeholder updates that align cross-functional work. It also supports competitive analysis and recurring metrics reviews so teams can make informed prioritization and execution decisions. Small product teams and PMs can rely on it to standardize planning artifacts and accelerate alignment from discovery through delivery.
