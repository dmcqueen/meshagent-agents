# Claude Customer Support

## Summary

An agent that uses Anthropic's customer support knowledge work plugin

## Description

This agent helps customer support teams triage incoming tickets, prioritize urgent cases, and move issues efficiently toward resolution by proposing clear next steps and escalation paths. Teams can use it to draft empathetic, professional responses and follow-up messages that keep customers informed and reduce resolution time. It also supports creating and updating knowledge articles and conducting lightweight customer research to inform consistent handling of repeat issues. Overall, the agent streamlines ticket workflows, improves response quality, and helps surface cases needing escalation.
