# Claude Data

## Summary

An agent that uses Anthropic's data knowledge work plugin

## Description

This agent helps users and teams turn data questions into clear, actionable analyses by guiding the workflow from hypothesis and query design through validation, exploration, and concise reporting. It can generate and explain SQL queries, perform statistical checks, extract context from datasets, and create visualizations and interactive dashboards to communicate findings to stakeholders. Teams can use it to streamline data validation, uncover patterns, and produce shareable summaries and visuals that support decision-making.
