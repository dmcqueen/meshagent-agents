# Claude Productivity

## Summary

An agent that uses Anthropic's productivity knowledge work plugin

## Description

This agent helps individuals and teams turn rough requests into structured plans and concrete deliverables, summarize context into clear next actions, and support day-to-day execution across writing, planning, and decision support. Users can rely on it to create organized task lists, manage priorities, extract key points from notes, and produce concise summaries that drive work forward. It is suited for improving memory and task management workflows, enabling faster progress on projects and clearer handoffs between collaborators. Brief runtime details: it operates as a productivity-focused conversational agent.
