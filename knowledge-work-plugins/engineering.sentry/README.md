# Claude Engineering Sentry

## Summary

Engineering worker that triages Sentry issues into Jira using incident-response and ticket-triage skills

## Description

This plugin supports engineering.sentry workflows and is designed to help users complete practical work with less manual effort. It can answer domain-specific questions in engineering.sentry work, turn ambiguous requests into actionable outputs, and adapt guidance to user goals, constraints, and context. Built-in guidance and reusable skills help keep outputs consistent, context-aware, and action-oriented. It works well for both quick answers and multi-step tasks that require planning, drafting, and refinement. Teams can use it as a day-to-day assistant for engineering.sentry decisions, analysis, and communication.
