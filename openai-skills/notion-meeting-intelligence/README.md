# OpenAI Notion Meeting Intelligence

## Summary

An agent that uses OpenAI's notion-meeting-intelligence skill and powered by gpt-5.2

## Description

This plugin supports notion-meeting-intelligence workflows and is designed to help users complete practical work with less manual effort. It can answer domain-specific questions in notion-meeting-intelligence work, turn ambiguous requests into actionable outputs, and adapt guidance to user goals, constraints, and context. Built-in guidance and reusable skills help keep outputs consistent, context-aware, and action-oriented. It works well for both quick answers and multi-step tasks that require planning, drafting, and refinement. Teams can use it as a day-to-day assistant for notion-meeting-intelligence decisions, analysis, and communication.
