# OpenAI Spreadsheet

## Summary

An agent that uses OpenAI's spreadsheet skill and powered by gpt-5.2

## Description

This agent lets individuals and teams work directly with spreadsheet-related tasks by answering domain-specific questions, clarifying ambiguous requests into concrete actions, and providing step-by-step guidance tailored to user goals and constraints. Users can rely on it to transform high-level objectives into practical spreadsheet outputs, refine formulas and workflows, and adapt recommendations to the context of their data and desired outcomes. It is suited for collaborative scenarios where iterative clarification and actionable directions are needed to complete spreadsheet work efficiently. The agent runs with a modern conversational model to produce clear, usable instructions and results.
