# OpenAI Jupyter Notebook

## Summary

An agent that uses OpenAI's jupyter-notebook skill and powered by gpt-5.2

## Description

This agent helps users and teams work effectively with Jupyter Notebook tasks by answering domain-specific questions, turning ambiguous requests into concrete, actionable steps, and adapting guidance to usersâ€™ goals and constraints. It is useful for clarifying notebook workflows, producing step-by-step instructions for analyses or notebook setups, and helping translate high-level objectives into reproducible notebook actions. Teams can use it to accelerate collaboration by getting clear, context-aware recommendations and troubleshooting advice tailored to their work in Jupyter Notebooks. The agent operates interactively with a model runtime to generate these practical outcomes.
