# OpenAI Gh Address Comments

## Summary

An agent that uses OpenAI's gh-address-comments skill

## Description

This agent helps teams manage and act on GitHub comment-related tasks by turning ambiguous or high-level requests into clear, actionable steps for addressing comments, resolving feedback, and coordinating follow-up work. Users can ask domain-specific questions about comment threads, get prioritized action lists, and receive concise summaries or suggested replies to streamline review and collaboration.
