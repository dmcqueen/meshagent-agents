# OpenAI Sentry

## Summary

An agent that uses OpenAI's sentry skill

## Description

This agent helps teams work with Sentry by answering domain-specific questions, converting unclear requests into clear, actionable steps, and tailoring guidance to the user’s goals, constraints, and context. Users can rely on it to troubleshoot issues, plan remediation workflows, and get stepwise recommendations that fit their project needs. It supports collaboration by producing concrete outputs teams can act on, such as prioritized tasks, investigation checklists, and concise explanations of Sentry concepts. The agent runs as a conversational assistant focused on Sentry-related work and adapts responses to the user’s situation.
