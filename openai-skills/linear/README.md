# OpenAI Linear

## Summary

An agent that uses OpenAI's linear skill

## Description

This agent enables users and teams to interact with a specialized linear-focused assistant that answers domain-specific questions, turns ambiguous requests into actionable outputs, and adapts its guidance to the user's goals, constraints, and context. It is useful for clarifying requirements, generating step-by-step plans, and producing concrete deliverables that advance linear-related tasks. Teams can rely on it to align work, resolve uncertainties, and produce consistent, goal-oriented recommendations. The agent runs with a capable conversational model to support iterative exchanges and refinement of outputs.
